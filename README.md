# 📦 VendorBridge

**VendorBridge** is a Procurement & Vendor Management ERP system designed to simplify and digitize end-to-end procurement operations. It provides a centralized platform to manage vendors, RFQs, quotations, purchase orders, approvals, invoices, and reporting workflows.

---

## 🚀 Features

VendorBridge helps organizations streamline procurement with powerful ERP capabilities:

* 🧾 Vendor Management (onboarding, profiles, performance tracking)
* 📩 RFQ (Request for Quotation) workflow
* 💬 Quotation comparison & approval system
* 🛒 Purchase Order creation & tracking
* 📊 Invoice generation & management
* 🔔 Approval workflows & notifications
* 📦 Inventory & procurement tracking
* 📈 Reporting & analytics dashboard
* 🔐 Role-based access control
* ⚡ Real-time procurement status updates

---

## 🎯 Project Goals

The vision of VendorBridge is to:

* Reduce manual procurement inefficiencies
* Centralize vendor communication
* Automate approval workflows
* Improve transparency in procurement cycles
* Provide real-time tracking of all purchasing activities

---

## 🏗️ Tech Stack

* Frontend: React / Vite
* Backend: Node.js / Express
* Database: PostgreSQL
* Authentication: JWT / Session-based auth
* Styling: Tailwind CSS

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/saloni375/Vendorbridge.git
cd Vendorbridge
```

### 2. Install dependencies

#### Frontend

```bash
cd frontend
npm install
```

#### Backend

```bash
cd backend
npm install
```

---

## ▶️ Running the Project

### Start Backend

```bash
cd backend
npm run dev
```

### Start Frontend

```bash
cd frontend
npm run dev
```

---

## 🧪 Environment Variables

Create a `.env` file in the backend:

```env
PORT=5000
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
```

---

## 📌 Common Issues

If you face errors like:

* `vite not recognized`
* `cors module not found`

Run:

```bash
npm install
npm install -g vite
npm install cors
```
